Privacy Policy for Run in Command

Introduction

Run in Command is a Windows utility that adds context menu functionality for Python files. This tool is designed to enhance your Python development experience by providing quick access to command prompt execution of Python scripts.

Registry Modifications

This tool interacts with your system in the following ways:

  • Modifies Windows Registry entries for context menu integration
  • Creates/removes registry keys under HKEY_CLASSES_ROOT
  • Requires administrator privileges for these modifications

Data Collection

Run in Command operates with complete privacy:

  • No personal information is collected
  • No data is stored or transmitted
  • No analytics or tracking is performed
  • Only performs local system registry modifications

System Changes

The tool makes the following system changes:

  • Adds a "Run in Command" entry to Python files' context menu
  • Creates necessary registry keys for functionality
  • All changes are reversible through the uninstall option

User Control

Users have complete control over the tool:

  • Install/uninstall the context menu entry at any time
  • All changes are transparent and documented
  • No background processes or services are installed
  • All modifications are contained to Python file associations

Security

Security measures include:

  • Administrator privileges required for modifications
  • No network access required
  • All operations are local to the system
  • No external dependencies beyond Python standard library

Contact Information

If you have any questions or concerns about this privacy policy, please reach out through GitHub.
