Usage Guide

Installation

  1. 1. Download the repository
  2. 2. Run requirements.py (optional)
  3. 3. Run run-in-command.py with administrator privileges
  4. 4. Select option 1 to add the context menu entry

Using the Tool

  1. 1. Right-click any Python file
  2. 2. Select "Run in Command" from the context menu
  3. 3. The script will execute in a Command Prompt window
  4. 4. Working directory is automatically set to the script's location

Uninstallation

  1. 1. Run run-in-command.py with administrator privileges
  2. 2. Select option 2 to remove the context menu entry
